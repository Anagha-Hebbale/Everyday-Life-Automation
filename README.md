# Automation Project
This project automates multiple tasks such as expense tracking, reminders, and restaurant recommendations using Python and external APIs.

The repository contains the source code, sample datasets, and input–output screenshots for reference.

## Project Structure

- `automation_code.py` – Main Python script  
- `expenses.csv` – Expense data file  
- `reminders.json` – Reminder storage file  
- `Bengaluru_Restaurants.csv` – Restaurant dataset  
- `Input-Output screenshots.pdf` – Program execution screenshots  
- `.gitignore` – Git ignore configuration  

Create a .env file Create a file named .env in the root directory of the project. 

Add your API keys in the following format: 

OPENWEATHER_API_KEY=your_api_key_here 

NEWS_API_KEY=your_api_key_here
